I'm hosted with GitHub Pages. You can reach me at newleon.github.io.

My intention was to create a github.io webpage for this existing repository.

The detailed guideline is given here: https://docs.github.com/en/pages/getting-started-with-github-pages/creating-a-github-pages-site

Basically you need to create two files first index.html and README.md. Add the some contents to both files. Click Settings under your depository name.

In the "Code and automation" section of the sidebar, click "Pages".

You need then to enable Github Pages and Save. Then a page will be autoamtically created with the URL newleon.github.io/repository_name or in this particular case newleon.github.io/App1min3d.
